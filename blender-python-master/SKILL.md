# Blender Python Master

Blender의 강력한 Python API (`bpy`)를 사용하여 3D 모델링, 애니메이션, 렌더링 작업을 자동화하는 스크립트를 작성합니다.

## 기능
*   **모델링 자동화**: 루프를 이용한 반복적인 오브젝트 생성, 배치.
*   **재질/텍스처 설정**: 노드 기반 쉐이더 자동 구성.
*   **씬 설정**: 카메라, 조명, 렌더링 세팅 자동화.

## 프롬프트 예시
*   "큐브 100개를 나선형으로 배치하는 스크립트 짜줘."
*   "선택한 오브젝트들에 랜덤한 메탈 재질을 입히는 bpy 코드 작성해줘."
*   "현재 씬의 모든 카메라를 찾아서 해상도를 1920x1080으로 변경해줘."

## 주의사항
*   작성된 코드는 Blender의 **Scripting 탭**에서 실행해야 합니다.
*   Blender 버전에 따라 API가 다를 수 있으므로, 타겟 버전을 명시하면 좋습니다 (예: Blender 4.0).
