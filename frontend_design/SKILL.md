# Frontend Design 🎨

웹사이트와 앱의 UI/UX를 아름답게 디자인하는 스킬입니다.

## 디자인 원칙

1. **모던 & 프리미엄**: 글래스모피즘, 그라디언트, 그림자 활용
2. **다크모드 지원**: 밝은/어두운 테마 모두 제공
3. **반응형 디자인**: 모바일/태블릿/데스크톱 대응
4. **애니메이션**: 부드러운 전환 효과

## 컬러 팔레트 예시

| 용도 | 라이트 모드 | 다크 모드 |
|------|-------------|-----------|
| 배경 | #FFFFFF | #1A1A2E |
| 텍스트 | #333333 | #E0E0E0 |
| 강조 | #6366F1 | #818CF8 |
| 성공 | #10B981 | #34D399 |

## CSS 스타일 기본값

```css
/* 글래스모피즘 */
backdrop-filter: blur(10px);
background: rgba(255, 255, 255, 0.1);
border-radius: 16px;

/* 부드러운 그림자 */
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);

/* 부드러운 전환 */
transition: all 0.3s ease;
```
