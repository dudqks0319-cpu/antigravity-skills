# Code Review ✅

코드 품질을 점검하고 개선점을 제안하는 스킬입니다.

## 리뷰 체크리스트

### 1. 코드 품질
- [ ] 변수명이 명확한가?
- [ ] 함수가 하나의 역할만 하는가?
- [ ] 중복 코드가 없는가?

### 2. 보안
- [ ] 민감 정보가 노출되지 않는가?
- [ ] 입력값 검증이 되어 있는가?

### 3. 성능
- [ ] 불필요한 연산이 없는가?
- [ ] 최적화 여지가 있는가?

## 리뷰 출력 형식

```markdown
## 코드 리뷰 결과

### ✅ 잘된 점
- ...

### ⚠️ 개선 제안
- 파일명:라인 - 개선 내용

### 📌 수정 권장
- ...
```

## 비개발자용 설명

- 전문 용어 사용 시 쉬운 비유로 설명
- "이 부분은 ~하면 더 좋아요" 형태로 안내
