# Brand Guidelines 🎯

프로젝트 전반에 걸쳐 일관된 브랜드 디자인을 유지하는 스킬입니다.

## Youngbeen 브랜드 가이드

### 컬러 팔레트

| 용도 | 색상 코드 | 설명 |
|------|-----------|------|
| Primary | #6366F1 | 인디고 (주요 강조) |
| Secondary | #8B5CF6 | 바이올렛 (보조) |
| Success | #10B981 | 그린 (성공) |
| Warning | #F59E0B | 앰버 (경고) |
| Error | #EF4444 | 레드 (오류) |

### 타이포그래피

| 용도 | 폰트 | 크기 |
|------|------|------|
| 제목 | Pretendard Bold | 24-32px |
| 본문 | Pretendard Regular | 16px |
| 캡션 | Pretendard Light | 14px |

### 디자인 원칙

1. **심플함**: 불필요한 장식 최소화
2. **일관성**: 동일한 스타일 반복 사용
3. **접근성**: 충분한 대비, 명확한 레이블
4. **현대적**: 둥근 모서리, 부드러운 그림자

> 💡 새 프로젝트 시작 시 이 가이드라인을 참조하여 일관된 디자인 적용
