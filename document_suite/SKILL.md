# Document Suite 📄

문서 파일을 생성하고 편집하는 스킬입니다.

## 지원 형식

| 형식 | 확장자 | 용도 |
|------|--------|------|
| Word | `.docx` | 보고서, 기획서 |
| Excel | `.xlsx` | 데이터 정리, 표 |
| PowerPoint | `.pptx` | 발표자료 |
| PDF | `.pdf` | 최종 문서 |

## 워크플로우

1. **형식 확인**: 사용자가 원하는 문서 형식 파악
2. **내용 구성**: 제목, 본문, 표 등 구조 설계
3. **파일 생성**: 해당 형식으로 파일 생성
4. **저장 위치**: `~/Desktop/프로젝트명/` 폴더에 저장

## 예시

```
사용자: "회의록을 Word로 만들어줘"
→ ~/Desktop/프로젝트명/회의록.docx 생성
```
