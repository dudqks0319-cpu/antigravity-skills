# Artifacts Builder 🏗️

웹 아티팩트(HTML, CSS, JavaScript)를 체계적으로 빌드하는 스킬입니다.

## 프로젝트 구조

```
~/Desktop/[프로젝트명]/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
└── assets/
    └── images/
```

## 빌드 워크플로우

1. **요구사항 분석**: 기능 목록 정리
2. **구조 설계**: 파일/폴더 구조 계획
3. **HTML 작성**: 시맨틱 마크업
4. **CSS 스타일링**: 반응형 + 다크모드
5. **JS 로직**: 인터랙션 구현
6. **미리보기**: 브라우저에서 확인
7. **저장**: 바탕화면 폴더에 저장

## 코드 품질 기준

- HTML: 시맨틱 태그 사용, 접근성 고려
- CSS: BEM 네이밍, CSS 변수 활용
- JS: ES6+ 문법, 주석 포함
