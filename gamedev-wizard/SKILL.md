# GameDev Wizard

게임 엔진별 최적화된 코드를 작성하고, 게임 아키텍처 설계를 돕습니다.

## 지원 엔진
1.  **Unity (C#)**: MonoBehavior 생명주기, 코루틴, DOTS 등.
2.  **Unreal Index (C++/Blueprints)**: Actor, Component, UPROPERTY 매크로 등.
3.  **Godot (GDScript)**: Node, Scene, Signal 처리.

## 기능
*   **스크립트 생성**: "캐릭터 점프 로직 C#으로 짜줘"
*   **쉐이더 작성**: HLSL/GLSL 또는 쉐이더 그래프 로직 설명.
*   **시스템 설계**: 인벤토리 시스템, 퀘스트 시스템 구조 제안.

## 프롬프트 예시
*   "Unity에서 오브젝트 풀링 매니저 스크립트 만들어줘."
*   "Unreal 5에서 C++로 커스텀 액터 컴포넌트 만드는 기본 구조 알려줘."
*   "Godot 4.0 캐릭터 이동 스크립트 작성해줘."
